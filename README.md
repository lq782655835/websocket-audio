# 环境准备

本Demo旨在探索使用Websocket实时传输音频流信息，供服务读取使用。使用[websocketd](https://github.com/joewalnes/websocketd/blob/master/README.md)服务器

## 安装websocketd

``` shell
brew install websocketd
```

## 运行websocket服务

``` shell
npm run dev
```

## 测试demo

双击打开demo.html(推荐使用chrome)
